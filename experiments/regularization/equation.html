• V : Vocabulary size
  • N : Number of autoencoders, which is one more than the number of transformer blocks
  • y i : True label for the token, which is set 1 for the correct token and 0 otherwise
  • p i : The predicted probability for the i -th token
  • ∥ X i ∥ : L2 norm of residual stream activations for the i -th layer
  • L reconstruct i : Reconstruction loss for the i -th encoder
  • λ reconstruct i : Reconstruction regularization term for the i -th encoder
  • L sparcity i : Feature sparsity loss for the i -th encoder
  • λ sparcity i : Sparsity regularization term for the i -th encoder
